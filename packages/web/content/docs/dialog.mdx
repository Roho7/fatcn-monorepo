## Design Guidelines

- Use a clear title and concise content to communicate the purpose.
- Provide actionable buttons (e.g., Confirm, Cancel) with clear hierarchy.
- Ensure dialogs are dismissible via a close button, overlay click, or `Esc` key.
- Center dialogs visually and use an overlay to focus user attention.
- Avoid overusing dialogs to prevent disrupting the user experience.

## Accessibility

- Supports headers, content areas, and footers.
- Includes primary, secondary, and dismiss actions.
- Fully navigable with the `Tab` key and dismissible with `Esc`.
- Adapts to various screen sizes and orientations.
- Uses roles like `dialog` and manages focus for accessibility.
