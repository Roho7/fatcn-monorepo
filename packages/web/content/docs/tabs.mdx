# Tabs

Tabs organize content into multiple sections, allowing users to navigate between them without leaving the page.

## Usage

<ComponentPreview name='tabs' />

## Props

| Prop        | Type                                                         | Description                                                |
| ----------- | ------------------------------------------------------------ | ---------------------------------------------------------- |
| `className` | `string`                                                     | The className to be passed to the tabs components (can be applied to each of the tabs components)          |

## Design Guidelines

- Use clear, descriptive labels for each tab to convey its content.
- Ensure the active tab is visually distinct and easy to identify.
- Maintain consistent spacing and alignment for tab labels.
- Keep content panels lightweight and directly related to the selected tab.
- Allow keyboard navigation with `Tab` and arrow keys, and visually indicate focus.

## Accessibility

- Switches between content sections seamlessly.
- Supports active, hover, and disabled states.
- Horizontal, vertical, or collapsible tabs.
- Fully navigable via keyboard shortcuts.
- Adapts gracefully to various screen sizes and orientations.
