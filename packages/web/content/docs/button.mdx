# Button

A button component that follows best practices for accessibility and provides multiple variants for different use cases.

## Usage

<ComponentPreview
name='button'
/>


## Features

- Multiple variants and sizes
- Accessible by default with ARIA attributes
- Support for icons and loading states
- Keyboard navigation support

## Examples

### Variants

Buttons come in different variants to match your design needs.

```tsx
<Button variant="primary">Primary</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="outline">Outline</Button>
<Button variant="ghost">Ghost</Button>
<Button variant="link">Link</Button>
```


### Sizes

Available in different sizes to fit your layout.


```tsx
<Button size="sm">Small</Button>
<Button size="md">Medium</Button>
<Button size="lg">Large</Button>
```

### States

Buttons support various states including disabled and loading.


```tsx
<Button disabled>Disabled</Button>
<Button loading>Loading</Button>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'primary' \| 'secondary' \| 'outline' \| 'ghost' \| 'link'` | `'primary'` | The visual style of the button |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | The size of the button |
| `disabled` | `boolean` | `false` | Whether the button is disabled |
| `loading` | `boolean` | `false` | Shows a loading spinner and disables the button |
| `asChild` | `boolean` | `false` | Merge props onto child element instead of rendering button |
| `className` | `string` | - | Additional CSS classes |

## Design Guidelines

- Use primary buttons for main actions
- Secondary buttons for alternative actions
- Outline buttons for less prominent actions
- Ghost buttons for subtle actions within a busy interface
- Link buttons for navigation-like actions

## Accessibility

The Button component follows WAI-ARIA guidelines:

- Uses native `<button>` element by default
- Properly handles disabled states
- Includes appropriate ARIA attributes
- Supports keyboard navigation
- Maintains sufficient color contrast
