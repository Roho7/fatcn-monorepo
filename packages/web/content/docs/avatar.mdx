### Sizes

Available in different sizes to fit your layout.

```tsx
<Avatar size="sm">Small</Avatar>
<Avatar size="md">Medium</Avatar>
<Avatar size="lg">Large</Avatar>
```
### Shapes

Available in different shapes to fit your layout.

```tsx
<Avatar shape="circle" src='https://your-img-url.com' alt='image'/>
<Avatar shape="square" src='https://your-img-url.com' alt='image'/>
```

## Props

| Prop        | Type                                                         | Default     | Description                                                |
| ----------- | ------------------------------------------------------------ | ----------- | ---------------------------------------------------------- |
| `size`      | `'sm' \| 'md' \| 'lg'`                                       | `'md'`      | The size of the button                                     |
| `shape`     | `'circle' \|'square'`                                        | `'primary'` | The shape of the button                                    |
| `src`       | `string`                                                     | -           | The url of the avatar inage                                |
| `alt`       | `string`                                                     | -           | Shows alternative string of the avatar                     |
| `ring`      | `boolean`                                                    | `false`     | Adds a border around the avatar                            |
| `asChild`   | `boolean`                                                    | `false`     | Merge props onto child element instead of rendering button |
| `className` | `string`                                                     | -           | Additional CSS classes                                     |

## Design Guidelines

- Use primary buttons for main actions
- Secondary buttons for alternative actions
- Outline buttons for less prominent actions
- Ghost buttons for subtle actions within a busy interface
- Link buttons for navigation-like actions

## Accessibility

The Button component follows WAI-ARIA guidelines:

- Multiple shapes and sizes
- Displays a profile image
- Shows initials when an image is unavailable
- Support for loading states
- Keyboard navigation support
