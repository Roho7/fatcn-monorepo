## Props

| Prop        | Type                                                          | Default     | Description                                                |
| ----------- | ------------------------------------------------------------- | ----------- | ---------------------------------------------------------- |
| `variant`   | `'default' \| 'secondary' \| 'warning' \| 'error' \| 'success`| `'default'` | The visual style of the callout                            |
| `children`  | `React.ReactNode`                                             | -           | Additional CSS classes                                     |

## Design Guidelines

- Use concise text to communicate the message.
- Use meaningful icons that align with the context of the message.
- Align text and visuals consistently for a clean appearance.
- Provide adequate padding and margin to make the callout easy to read.
- Use a uniform style for callouts across the interface to maintain coherence.

## Accessibility

- Highlight essential information only.
- Apply color intentionally (e.g., green for success).
- Position callouts strategically for visibility.
- Ensure accessibility with ARIA and keyboard support.
- Test across devices for effectiveness.
